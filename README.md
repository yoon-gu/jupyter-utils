# jupyter-utils

폐쇄망 + 타 조직 제공 Jupyter 환경이라는 **제약 위에서 최대한의 효율**을 내기 위한
Jupyter Lab 유틸리티 모음입니다. (함수·매직 커맨드·확장·스니펫 등)

## 환경 제약 (Context)

이 저장소의 모든 결과물은 아래 제약을 전제로 합니다. 이슈/구현 시 항상 염두에 둬주세요.

- 🚫 **폐쇄망**: 인터넷에서 `pip install`로 새 패키지를 받기 어렵습니다. 표준 라이브러리와 이미 설치된 패키지 위주로 동작해야 합니다.
- 🔒 **권한 제약**: 관리자/sudo 권한이 없습니다. 설정은 보통 홈 디렉터리(`~/.ipython`, `~/.jupyter`) 한정입니다.
- 🔁 **서버 재시작 불가**: Jupyter 서버는 타 조직이 띄워줍니다. 커널 재시작 정도만 가능합니다.
- 👥 **공유 환경**: 여러 사람이 함께 쓰므로 전역 변경은 지양합니다.

설치가 필요 없는 순서로 선호합니다: **스니펫 → 매직 커맨드 → 함수/모듈 → (가능하면) 확장**.

## 워크플로우

1. 틈날 때 [이슈 템플릿](.github/ISSUE_TEMPLATE)으로 아이디어를 기록합니다.
2. 각 이슈에는 **환경 제약 체크박스**와 **완료 조건(Acceptance Criteria)**을 적어둡니다.
3. 나중에 Claude Code / Codex 같은 AI 에이전트에게 이슈를 넘겨 구현합니다.
   - 완료 조건이 곧 작업 지시문이 됩니다.

## 이슈 템플릿

| 템플릿 | 용도 |
| --- | --- |
| 🧰 새 유틸리티 / 기능 제안 | 함수·모듈 등 새 도구 제안 |
| ✨ IPython 매직 커맨드 | `%`/`%%` 매직 추가·개선 |
| 🧩 Jupyter 확장 / 환경 설정 | 확장·CSS/JS·키바인딩·config |
| 📋 스니펫 / 레시피 | 복붙용 코드 스니펫 정리 |
| 🐞 버그 리포트 | 기존 결과물 오동작 |
| 💭 빠른 아이디어 메모 | 한 줄 메모(나중에 구체화) |

> 참고: 템플릿이 참조하는 라벨(`utility`, `magic`, `extension`, `snippet`, `idea`,
> `needs-triage` 등)은 저장소 Labels에 미리 만들어두면 자동으로 붙습니다.
